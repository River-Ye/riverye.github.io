Ubuntu 的游標消失，該怎辦?

  1. 將 Windows 預設輸入模式從「中文模式」改為「英數模式」，並重開 Ubuntu。

  2. 在 Ubuntu 中若不小心切換到中文模式並輸入任意鍵時，游標又會消失，解決方法為在 Ubuntu 中輸入任意中文字後並按確認 (並非送出指令喔)，此時游標再次出現，接著切換回「英數模式」即可。 (中英文切換快捷鍵：Shift 按一下)
